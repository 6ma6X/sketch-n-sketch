Sketch-n-Sketch

Programming + Direct Manipulation + HTML/SVG

Direct manipulation interfaces are useful in many domains, but the lack of programmability in a high-level language makes it difficult to develop complex and reusable content. We envision direct manipulation programming systems that allow users the benefits of general-purpose programming languages but with less keyboard-based text editing and more mouse-based direct manipulation. Sketch-n-Sketch is a direct manipulation programming system for creating HTML and SVG documents.
